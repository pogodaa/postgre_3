# Обеспечение защиты базы данных и контроль целостности информации

## 📋 Описание
Учебный проект по настройке безопасности СУБД PostgreSQL. Реализована система разграничения доступа, 
аудит действий пользователей и контроль целостности данных.

## 🗄️ Структура базы данных
| Таблица | Описание |
|---------|----------|
| `users` | Пользователи системы |
| `categories` | Категории транзакций |
| `transactions` | Финансовые операции |
| `audit_log` | Журнал аудита изменений |

## 🔐 Роли и права доступа
| Роль | Пользователь | Права |
|------|--------------|-------|
| `accountant` | `user_accountant` | SELECT, INSERT, UPDATE (transactions) |
| `auditor` | `user_auditor` | SELECT (все таблицы + логи) |
| `manager` | `user_manager` | ALL (полный доступ) |

## 🛡️ Меры безопасности
- ✅ Разграничение прав доступа (ACL)
- ✅ Триггеры для аудита изменений
- ✅ CHECK-ограничения (целостность данных)
- ✅ Резервное копирование (pg_dump)
